## Contents

The repository includes the following Java programs:

* **adddigitsinnum.java** – Adds the digits of a given number
* **avgofNnum.java** – Calculates the average of N numbers
* **factorial.java** – Computes the factorial of a number
* **fibonacciseries.java** – Generates the Fibonacci series
* **findprimenums.java** – Finds prime numbers
* **palindromenumber.java** – Checks whether a number is a palindrome
* **reversestring.java** – Reverses a given string
* **sumandavgofarray.java** – Finds the sum and average of array elements
* **sumofall.java** – Calculates the sum of numbers
* **sumoffirstnnumbers.java** – Computes the sum of the first N natural numbers

## How to Run the Programs

1. Make sure Java is installed on your system

   ```bash
   java --version
   ```

2. Clone the repository

   ```bash
   git clone https://github.com/yashmishra11/small-java-programs.git
   ```

3. Navigate to the project directory

   ```bash
   cd small-java-programs
   ```

4. Compile a program

   ```bash
   javac filename.java
   ```

5. Run the program

   ```bash
   java filename
   ```
